== System Requirements
Windows:
  Windows 7, Vista, XP
  Visual Studio
Mac/Linux:
  CMake
  CMake-compatible build tools (e.g. GCC and GNU Make)
All:
  OpenGL and GLUT libraries

== Compiling
This project uses CMake to generate a Makefile which can then be used to build
the binaries. A Visual Studio project is also included for compatibility with
the Windows platform.
